RPI WEATHER

Raspberry Pi based weather forecast display.
View on GitHub

Displays the weather forecast for the next four days as icons on 8x8 LED matrices. Based on a Raspberry Pi using LED matrices from Adafruit. Housed in a custom built wood enclosure with a glass front panel.

HARDWARE

Here are some details on the wooden case used for this project. The overall parts are shown below.

The LED matrices are held in place using particle board with cut outs for each matrix. This sits in slots behind the glass.

The front of the LED panel is painted black to minimize the potential for any light to reflect back out.

The LED panel and glass sit in slots.

A Fritzing file is included in the repo which shows the wiring connections.

Everything gets hooked together just before closing it all up.

And then it's ready to go.
